Some Recent Projects

BetterVision Glasses

For my Intro to Engineering Design's semester-long design project, I worked on a team of 4 to create a pair of affordable assistive glasses for the visually impaired.

We modeled custom frames in Fusion360 and 3D printed the results. We then mounted an Arduino Nano and Ultrasonic sensors to the frame and used them to detect the distance and angle of objects in front of the user. This information was then transmitted to a pair of Bluetooth headphones.

I wrote all the code in the Arduino language (similar to C++) for the ultrasonic sensors and the Bluetooth module.

Card Reader Classifier

As a software intern this past summer, I was tasked with creating a RFID card reader classifier for easily identifying the specific make and model of a badge scanner for easier integration with Oloid's proprietary keyless scanner.

I creating a Tensorflow pipeline with dropout regularization and utilized data autotuning and data augmentation to improve the model's accuracy. I used React.js to create a front-end UI that allows users to upload images of RFID card readers and receive the model predictions.
